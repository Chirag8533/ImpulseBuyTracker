# ImpulseTrack

ImpulseTrack is a web-based application designed to help users track and analyze their impulse purchases. It allows users to log items, monitor spending behavior, and reflect on purchase decisions over time.

## 🚀 Features
- Log purchases with item name, amount, category, and date
- Track emotional response (Worth it / Regret / Unsure)
- View total spending and number of purchases
- Calculate regret rate based on user inputs
- Filter purchases based on mood
- Persistent data storage using localStorage

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript (Vanilla JS)

## 💡 Key Highlights
- Dynamic DOM manipulation for real-time updates
- State management using JavaScript arrays
- Data persistence using browser localStorage
- Clean and responsive UI using modern CSS (Flexbox & Grid)

## 🎯 Purpose
The goal of this project is to promote mindful spending by helping users reflect on their buying habits and identify patterns in impulse purchases.

## 📌 Future Improvements
- Add data visualization (charts)
- Monthly spending analysis
- User authentication and backend integration
